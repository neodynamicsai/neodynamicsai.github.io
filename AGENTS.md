# Repository Rules

## Messaging Consistency
- Keep all public-facing metadata aligned with the current company positioning and pitch deck language.
- When messaging changes, update `index.html` tags together: `title`, `meta[name="description"]`, `meta[name="keywords"]`, Open Graph (`og:title`, `og:description`), and Twitter (`twitter:title`, `twitter:description`).
- Do not leave mixed/legacy wording across these tags.
